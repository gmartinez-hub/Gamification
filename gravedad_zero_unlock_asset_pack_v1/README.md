# Gravedad Zero Unlock Asset Pack v1

Este pack contiene assets visuales + configs + briefs para el evento de reliquia/holograma que desbloquea cambios de stage.

## Integración sugerida en repo

Copiar carpetas:

```txt
assets/hologram/
assets/vfx/
config/ o src/config/
docs/
```

Recomendación: no integrar directo en `main` mientras Codex esté trabajando. Usar branch separado o pasarle el ZIP para que lo integre en su branch actual.

## Contenido

- PNG green screen original del holograma.
- PNG alpha full/cropped.
- anillos holográficos.
- aura/glow.
- scanlines.
- particles atlas.
- shockwave ring.
- flash glow.
- cracks overlay para obstáculo grande.
- configs JSON.
- briefs Markdown.
