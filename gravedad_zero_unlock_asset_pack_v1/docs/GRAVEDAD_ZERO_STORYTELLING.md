# Gravedad Zero — Storytelling v1

## Concepto

Una nave contra el caos del espacio.

Cada obstáculo superado libera una señal.
Cada señal revela una ruta.
Cada ruta acerca la nave al final del camino.

Toma el control.

## Tono

- Narrativa en español.
- Microcopy arcade en inglés.
- No mencionar comercio, ventas, cobros, facturación, cliente ni negocio dentro de la capa de juego.

## Boot copy

```txt
GRAVEDAD ZERO

RUTA DESCONOCIDA

TOMA EL CONTROL
```
