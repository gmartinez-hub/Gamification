# QA Master Checklist — Gravedad Zero

## Mission 01

- [ ] Inicia en Stage 1.
- [ ] Se activan 3 asteroides chicos.
- [ ] Astronauta puede romperlos.
- [ ] Contador llega a 3/3.
- [ ] Aparece obstáculo grande.
- [ ] Nave puede destruirlo.
- [ ] Aparece reliquia.
- [ ] Astronauta toca reliquia.
- [ ] Se dispara energy transfer.
- [ ] Se muestra STAGE UNLOCKED.
- [ ] Cambia a Stage 2 con transición existente.

## Aim Assist

- [ ] Click en target válido activa reticle.
- [ ] Click inválido no activa slow motion.
- [ ] Slow motion dura menos de 0.8s.
- [ ] Fire cue coincide con audio.
- [ ] Heavy shot se siente más fuerte que tool pulse.

## Robot Companion

- [ ] Aparece arriba/derecha.
- [ ] Click abre/cierra panel.
- [ ] Panel muestra objetivos pendientes.
- [ ] Cambia a alert ante obstáculo grande.
- [ ] Cambia a stage_clear al unlock.
- [ ] No tapa gameplay.

## Menú

- [ ] Fondo negro.
- [ ] Letras blancas simples.
- [ ] Acentos mínimos cyan/magenta.
- [ ] Briefing se entiende.
- [ ] Controles se entienden.
