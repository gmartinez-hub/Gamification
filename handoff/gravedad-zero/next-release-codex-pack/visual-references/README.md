## 01_companion_reference_correct.png

Referencia principal del companion.

Usar este look:
- esférico;
- blanco/perlado;
- lentes violetas/cyan;
- antenas con tips naranjas;
- limpio;
- 3D;
- sin sombra/base/halo.
