# 02 — Release Requirements

## Resultado esperado

La próxima versión debe sentirse como una demo más cerrada:

```txt
- companion correcto y usable;
- autoaim premium;
- mundo navegable grande;
- más vida espacial;
- velocidad con progresión;
- audio que acompaña la velocidad;
- tutorial integrado desde el companion.
```

## Prioridades

### P0 — Bloqueantes

```txt
Companion visual correcto
Autoaim rota modelo
Eliminar spray raro/líneas raras
Mapa grande/procedural visible
Botón de velocidad
Companion tutorial
```

### P1 — Importantes

```txt
Más planetas y cuerpos sintéticos
Assets Three realmente usados
Velocidad por stage
Audio/música por velocidad
QA de 3 min por dirección
```

### P2 — Polish

```txt
Pequeños transitions
Microcopy
Paneles más lindos
Pequeños pings
```

## No hacer

```txt
No agregar features narrativas nuevas.
No generar más nebulosas pesadas.
No meter más overlays random.
No cambiar la identidad del companion.
No crear otro personaje.
No llenar el mundo de debris chico.
```
