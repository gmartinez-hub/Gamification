# 05 — Gameplay Unlocks, Turbo and Aim

## Unlocks

```txt
Gema 0: Sector 1 + turbo limitado.
Gema 1: Sector 2 + Turbo x2 + nuevos mundos mecánicos.
Gema 2: Sector 3 + Turbo x3 + mundo más inestable.
Gema 3: Final + Warp Pulse + cuerpos reliquia.
```

## Turbo

```txt
F = mantener turbo.
```

Debe tener:

```txt
- propulsión/fuego/energy jet;
- audio boost;
- parallax más rápido;
- cámara sutil;
- límite por gema/stage.
```

## Autoaim

No puede resolver desde cualquier lugar.

Debe tener:

```txt
- rango;
- modo correcto;
- % de éxito;
- desvío/miss;
- feedback de fuera de rango;
- rotación real o campo de orientación.
```

## Field fallback

Si rotar sprite/modelo no se percibe:

```txt
usar campo de orientación cerca del actor.
```

No encima del target.

## QA

```txt
[ ] F acelera.
[ ] Turbo se desbloquea.
[ ] Autoaim requiere rango.
[ ] Hay % de éxito.
[ ] Hay miss/desvío.
[ ] Hay sensación zero-g.
```
