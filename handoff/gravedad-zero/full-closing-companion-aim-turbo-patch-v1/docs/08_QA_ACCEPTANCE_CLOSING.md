# 08 — QA Acceptance Closing

## Companion

```txt
[ ] Comparado con reference, de frente parece el mismo personaje.
[ ] No hay cara procedural deformada.
[ ] No hay halo/sombra/base sucia.
[ ] Tiene idle/talk/alert/success.
```

## Aim

```txt
[ ] No es una línea instantánea.
[ ] Hay lock/stabilize/orient.
[ ] Proyectil viaja en cámara lenta.
[ ] Impacto usa atlas.
[ ] Miss viaja a missPoint.
[ ] Major target puede usar timing extendido/cinemático.
```

## Turbo

```txt
[ ] Funciona en 8 direcciones.
[ ] No se ve como spray.
[ ] Está alineado al movimiento.
[ ] Se entiende aceleración.
```

## Scale / orbits

```txt
[ ] Targets lejos.
[ ] Mapa más grande.
[ ] Stage 2 exige perseguir.
[ ] Stage 3 exige anticipar.
[ ] Planetas no recolorean.
```

## Evidencia requerida

```txt
- screenshot companion close-up vs reference.
- video aim normal.
- video aim cinematic/major.
- video turbo 8 direcciones.
- video stage 2 persiguiendo targets.
- video stage 3 anticipando movimiento.
- video planet identity: stage change sin recolor.
```
