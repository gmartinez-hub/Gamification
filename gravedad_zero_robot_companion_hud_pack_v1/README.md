# Gravedad Zero — Robot Companion HUD Pack v1

Pack para sumar el robot como companion/indicador de estado.

## Rol

- Vive arriba/derecha en gameplay.
- Se puede clickear.
- Al clickearlo muestra un cuadro con lo que falta.
- Cambia de estado según misión.
- Puede wrappearse en Three como sprite/billboard.
- El texto queda en HTML/CSS.
