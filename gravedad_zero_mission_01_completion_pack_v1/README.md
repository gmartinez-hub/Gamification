# Gravedad Zero — Mission 01 Completion Pack v1

Pack externo para terminar la primera misión sin pisar el trabajo activo de Codex.

## Incluye

- Assets visuales PNG para VFX.
- Assets del holograma/reliquia.
- Audios WAV limpios para la misión.
- Configs JSON.
- Helpers JS opcionales.
- Briefs y QA checklist.

## Integración sugerida

Copiar a un branch separado o pasarle el ZIP a Codex.

No commitear directo a `main` si Codex todavía está trabajando.

## Estructura

```txt
assets/audio/
assets/vfx/
assets/hologram/
config/
src/audio/
src/mission/
src/vfx/
src/ui/
docs/
```
